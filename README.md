<h1>ACOAPP</h1>

<p>ACOAPP is a mobile tool app that helps Aircraft Coordinators to manage calculations and observation data more precisely on the ground.</p>

<h2>To run:</h2>
<p>npm install -g ionic</p>
<p>npm install</p>

<p>ionic serve</p>
